Keyboard Focus Test

Keyboard navigation (and a visible focus indicator) is required section 508 accessibility compliance ( Part 2 - Section c) and for WCAG 2 Level AA compliance (WCAG 2.4.7). Keyboard navigation is required for all pages on http://www.nps.gov.

Try navigating this page without your mouse. Press the tab (or shift-tab) key to navigate the keyboard focus to each link (tab will also visit button elements and any element with a non-negative tabindex attribute if those existed on the page). Press the enter key to visit the link.
